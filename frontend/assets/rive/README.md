# Rive Animations

Bu klasöre Rive animasyon dosyaları (.riv) eklenmelidir.

## Gerekli Dosyalar

| Dosya | Animasyon | Kullanım |
|-------|-----------|----------|
| `success_check.riv` | Checkmark | Ödeme/İşlem başarılı |
| `error_x.riv` | X animasyonu | Hata durumları |
| `loading_dots.riv` | Yükleniyor | API bekleme |
| `empty_box.riv` | Boş kutu | Liste boş |
| `money_fly.riv` | Para uçuşu | Gelir gider |
| `house_pop.riv` | Ev pop-up | Yeni mülk eklendi |
| `chat_bubble.riv` | Mesaj balonu | Yeni mesaj |
| `pulse_ring.riv` | Pulse ring | Acil bildirim |

## Oluşturma

1. https://rive.app/app adresine gidin
2. Yeni animasyon oluşturun
3. export edin (.riv)
4. Bu klasöre kopyalayın
